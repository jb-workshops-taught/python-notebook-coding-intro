Variables

To save a number or word, we have to give it a name. The name is called a variable because its value can vary; it can change over time. Here are some variables and their values.

x = 10 y = 20 name = "Jenny" firstLetter = "J" howManyApples = 5

When you set a variable's value, that is called assigning a value to the variable.

The + symbol does addition and - does subtraction. What value does the total variable hold? Copy-paste into Jupyter and run it. Were you right?

total = 2 + 4 + 3 - 1 print(total)

The / symbol does division, like a fraction would. What value does the average variable hold? Copy-paste into Jupyter and run it. Were you right?

average = (5 + 6 + 6 + 3) / 4 print(average)

The * symbol does multiplication. In this example, a customer is buying 3 loaves of bread which cost $1.99 each. The sales tax is 5%. What is the total price of their order? Copy-paste into Jupyter and run it. Were you right?

howMany = 3 priceEach = 1.99 salesTaxPercent = 0.05 totalPrice = (howMany * priceEach) + (salesTaxPercent * (howMany * priceEach)) print(totalPrice)

The counter variable demonstrates that the variable's value can be reassigned: you can keep assigning new values to it. It remembers only the last value.

counter = 0 counter = 1 counter = 2 counter = 3 counter = 4 print(counter)

Here's another example of assigning new values. Can you guess what it prints? Try it.

counter = 7 counter = 5 counter = 100 print(counter) counter = 3 counter = -42 print(counter)

The next example uses String variables. String means text - letters, punctuation, words, and sentences. The name comes from "a string of letters" like an alphabet bead bracelet. A single letter or punctuation mark is called a character. A string is a way to put together many characters, to represent human text. Any value inside "double quote marks" is a string.

Run this code example. What does it do?

print("Good morning!") name = input("What is your name? ") print("Hi " + name + ". It's good to meet you!")
Note: If you get an error like "NameError: name 'Jenny' is not defined" then you are using Python 2. Fix this by choosing File, New, Python 3 Notebook. Or, change input to raw_input.
