While Loops

A while loop runs as long as the condition is true.

count = 0 while (count < 5): print "Still going! ", count count = (count + 1) print "All done!"

The indented code repeats as long as the comparison at the top of the loop remains true. When the comparison is no longer true, then the loop finishes, and the code after the loop runs.

Guess what the code above prints. Then run it in Jupyter and see if you're right.

You can also use a boolean variable to control the loop, and change the boolean variable's value inside the loop while it's running, to stop the loop. (You were introduced to boolean values in lesson 3, "If Statements" - review that if you are unsure.)

word = "" keepGoing = True print "Guess the magic word." while(keepGoing == True): word = input("What is the word? ") if (word == "magic"): print "We love magic!" elif (word == "quit"): keepGoing = False else: print "You typed ", word print "All done!"

Run that example. Try typing in the following words, in order:

sesame apple room magic buzz quit

Atalia wants to write an adding machine. As long as the user of the program keeps typing in numbers, the program will keep adding them up. When the user types in zero, then the program ends. How can she use a while loop to write this program? There are multiple ways to do this. The example below shows one of them.

sum = 0 number = 1 while (number != 0): number = input("How much to add? ") sum = sum + number print "Sum is ", sum

While Loops: Learning Goals

  1. Recognize when a loop is a useful strategy.
  2. Choose a meaningful comparison to control the loop.
  3. Understand how to apply a while loop to a real situation.
  4. Make good choices about what goes before the loop, inside the loop, and after the loop.

While Loops: Concepts Summary

  1. A while loop repeats the indented set of code as long as the comparison at the top remains true.
  2. The un-indented lines are not part of the loop.
  3. Pay attention to what condition triggers the end of the loop.
